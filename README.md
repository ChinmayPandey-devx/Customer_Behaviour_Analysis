This project focuses on analyzing customer purchasing behavior using data analytics techniques. The goal is to identify patterns, trends, and insights that help businesses understand their customers better and make data-driven decisions.

The dataset was cleaned, processed, and analyzed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn. Exploratory Data Analysis (EDA) was performed to uncover relationships between variables like customer demographics, product categories, purchase frequency, and spending patterns.

Key insights include identifying high-value customers, understanding category-wise purchasing trends, and analyzing factors that influence customer spending behavior. Visualizations and statistical analysis were used to present the findings clearly.

This project demonstrates practical skills in data cleaning, exploratory data analysis, data visualization, and deriving business insights from real-world datasets.
